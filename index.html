Jolly by sbarski

The Game of Life implemented with HMTL5 and JS

Jolly (inspired by Golly) is an online "Game of Life" simulator There are a number of features which differentiate Jolly from other online implementation of the game of life. Current Features: 1. Implements hashlife 2. Reads RLE and Cells files 3. UI (based on canvas) with zoom in/out
